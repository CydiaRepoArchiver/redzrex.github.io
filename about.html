What is?

Cydia

Cydia /sɪˈdi.ə/ is a software application for iOS
that enables a user to find and install software
packages on jailbroken iOS Apple devices such as
the iPhone, the iPod Touch, and the iPad.

Repositories

Repositories (also called sources) host packages.
You can think of Cydia a bit like a web browser:
consider that Safari doesn't contain or own Google
or Amazon, but instead is a way to view Google,
Amazon, and any other website. Cydia is a way to
browse and view software hosted by independent
repositories, and the Cydia Store is a purchasing
system that lets you buy software hosted by them.

Jailbreak

Jailbreaking is the process of removing hardware
restrictions on iOS, Apple's operating system,
on devices running it through the use of software
and hardware exploits; such devices include the
iPhone, iPod touch, iPad, and second-generation Apple TV.
